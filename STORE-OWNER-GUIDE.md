## üé® Customizing Your Theme

### Theme Editor (No Code Required!)

After uploading, customize everything in **Shopify Admin ‚Üí Online Store ‚Üí Themes ‚Üí Customize**:

#### 1. **Theme Settings** (Global Design)
Click "Theme settings" in the left sidebar to change:
- **Colors** - Pre-set to black/white/red vintage aesthetic
- **Typography** - Uses "Assistant" font
- **Layout** - Page width, spacing, borders
- **Buttons** - Already styled with 2px borders, sharp corners
- **Cart** - Set to drawer mode (slides in from side)

#### 2. **Header Settings**
Click the header section to edit:
- Upload your logo (replace placeholder)
- Choose which menu to display
- Enable/disable sticky header
- Adjust header padding

#### 3. **Footer Settings**
Click the footer section to edit:
- Change brand name and description
- Add your social media links (Instagram, Pinterest, TikTok)
- Customize footer menus (Shop, About, Help columns)
- Newsletter signup heading

#### 4. **Instagram Feed**
Click the Instagram Feed section:
- **Add Images**: Click "Add block" ‚Üí Select "Instagram Post"
  - Upload image
  - Add link to the Instagram post
  - Add caption/alt text
- Recommended: Add 6-12 posts for smooth scrolling
- Adjust scroll speed (default: 60 seconds)
- Change post size (default: 300px)

## üìù Important Setup Tasks

### Must Do Before Launch:

#### 1. Upload Your Logo
- Go to **Theme settings ‚Üí Logo**
- Upload your logo image (recommended: 240x80px PNG with transparent background)
- Or the header will use the fallback "FORGOTTEN ESSENTIALS" text

#### 2. Create Navigation Menus
- Go to **Shopify Admin ‚Üí Online Store ‚Üí Navigation**
- Create a menu called "Main Menu" for the header
- Add your key pages (Shop, New Arrivals, About, Contact)

#### 3. Set Up Collections
The theme will automatically display your collections. Make sure you have:
- Collection images uploaded
- Collection descriptions written
- At least 3-6 collections for the "Shop by Category" section

#### 4. Add Instagram Images
- The Instagram section has 6 placeholder slots
- Replace these with your actual Instagram images
- Each image needs to be uploaded in the Theme Editor

#### 5. Configure Social Links
- **Theme settings ‚Üí Social media**
- Add your Instagram URL (already set to @forgottenessentials)
- Add Pinterest, TikTok if you have them

## üÜò Troubleshooting

### Header/Footer Not Showing?
- Make sure you uploaded the complete theme ZIP
- Don't edit the section group files directly
- If missing, re-upload the theme

### Instagram Feed Empty?
- Add images in Theme Editor ‚Üí Instagram Feed section
- Click "Add block" ‚Üí "Instagram Post"
- Upload at least 3 images for it to display

### Cart Not Working?
- Go to Theme settings ‚Üí Cart
- Ensure "Cart type" is set to "Drawer"
- Save changes

### Products Not Displaying?
- Make sure products are assigned to collections
- Check that collections exist in your store
- Ensure products are set to "Active"
